Introduction

2002年、香川県高松市生まれ。
県内の高校卒業後に、香川大学創造工学部機械システムコースに入学。
4力をはじめとする機械系の基本的な事を学んでいたが、機械学習、制御工学を扱う際に必要なC言語について触れ、プログラミングに対する興味が湧く。その後、様々な言語について調べている際にWeb開発系について知り、2022年の夏から独学で学習を開始。
3ヶ月が経った頃からオンライン型プログラミングスクールであるデイトラの利用を開始。
現在は、フロントエンドエンジニアとして活躍するため、日々制作に励んでいる。

Skills
